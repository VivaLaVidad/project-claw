## 🚀 快速开始

### 第 1 步：初始化模型网关

```python
from cloud_server.model_gateway import initialize_model_gateway

# 初始化网关
gateway = await initialize_model_gateway()
```

### 第 2 步：执行推理

```python
from cloud_server.model_gateway import get_model_gateway
from cloud_server.cost_tracker import CostTracker, CostRecord, usd_to_cny
import uuid

gateway = get_model_gateway()
tracker = CostTracker()

# 执行推理
query = "推荐一碗拉面"
messages = [{"role": "user", "content": query}]

result, usage, cost = await gateway.route_and_infer(
    query=query,
    messages=messages,
    tenant_id="ramen_shop_001"
)

# 记录成本
record = CostRecord(
    transaction_id=str(uuid.uuid4()),
    tenant_id="ramen_shop_001",
    model_id=result.get("model", "unknown"),
    input_tokens=usage.input_tokens,
    output_tokens=usage.output_tokens,
    total_tokens=usage.total_tokens,
    cost_usd=cost,
    cost_cny=usd_to_cny(cost),
    timestamp=time.time(),
    query=query
)

tracker.record_cost(record)
```

### 第 3 步：查看成本报告

```python
# 获取租户成本
tenant_cost = tracker.get_tenant_cost("ramen_shop_001")
print(tenant_cost)

# 获取每日成本
daily_cost = tracker.get_daily_cost("2024-01-01")
print(daily_cost)

# 获取模型成本分布
model_breakdown = tracker.get_model_cost_breakdown("ramen_shop_001")
print(model_breakdown)
```

## 💰 成本计算

### 成本公式
```
输入成本 = (输入 Token 数 / 1000) × 每 1000 Token 的成本
输出成本 = (输出 Token 数 / 1000) × 每 1000 Token 的成本
总成本 = 输入成本 + 输出成本
```

### 成本示例
```
查询: "推荐一碗拉面"
模型: qwen-1.8b (本地)
输入 Token: 10
输出 Token: 50
总成本: $0.0

查询: "我想要一碗特别便宜的拉面，能不能打折？"
模型: deepseek-chat (API)
输入 Token: 20
输出 Token: 100
总成本: $0.0014 × (20/1000) + $0.0042 × (100/1000) = $0.00028 + $0.00042 = $0.0007
```

## 🛡️ 故障转移机制

### 故障转移流程

```
1. 尝试主模型
   ↓
2. 如果失败 (5xx 或超时)
   ↓
3. 尝试第一个备用模型
   ↓
4. 如果失败
   ↓
5. 尝试第二个备用模型
   ↓
6. 如果全部失败
   ↓
7. 返回错误
```

### 故障检测
```python
# 5xx 错误
if response.status_code >= 500:
    # 触发故障转移

# 超时
except asyncio.TimeoutError:
    # 触发故障转移
```

```python
import asyncio
from cloud_server.model_gateway import initialize_model_gateway, get_model_gateway
from cloud_server.cost_tracker import CostTracker, CostRecord, usd_to_cny
import uuid, time

async def main():
    # 初始化
    await initialize_model_gateway()
    gateway = get_model_gateway()
    tracker = CostTracker()
    
    # 执行推理
    query = "推荐一碗拉面"
    result, usage, cost = await gateway.route_and_infer(
        query=query,
        messages=[{"role": "user", "content": query}],
        tenant_id="ramen_shop_001"
    )
    
    # 记录成本
    record = CostRecord(
        transaction_id=str(uuid.uuid4()),
        tenant_id="ramen_shop_001",
        model_id=result.get("model", "unknown"),
        input_tokens=usage.input_tokens,
        output_tokens=usage.output_tokens,
        total_tokens=usage.total_tokens,
        cost_usd=cost,
        cost_cny=usd_to_cny(cost),
        timestamp=time.time(),
        query=query
    )
    tracker.record_cost(record)
    
    # 查看报告
    print(tracker.get_tenant_cost("ramen_shop_001"))

asyncio.run(main())
```
