# Project Claw v9.0 - 工业级龙虾自动回复系统

## 项目概述

Project Claw 是一个完整的工业级自动回复系统，集成了 OCR、气泡检测、消息去重、LLM 回复、飞书同步等功能。

---

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置环境

编辑 `.env` 文件，填入你的 API 密钥。

### 3. 运行程序

```bash
python lobster_mvp.py
```

---

## 核心功能

### 消息识别流程

1. **截图**：每 0.5 秒截一次屏幕
2. **裁剪**：只识别聊天主体区域
3. **OCR**：使用 EasyOCR 识别文本
4. **清洗**：过滤噪声文本
5. **气泡检测**：判断是否为用户消息
6. **位置追踪**：检测消息是否为新消息
7. **去重**：基于内容 hash 和时间窗口去重

### 回复流程

1. **LLM 调用**：使用 DeepSeek API 生成回复
2. **输入**：点击输入框，输入回复内容
3. **发送**：按 Enter 键发送
4. **同步**：异步同步到飞书群和多维表格

---

## 工业级特性

### 1. 位置追踪
- 记录上一条消息的 bbox 位置
- 只有位置变化超过 50 像素才认为是新消息
- 防止同一条消息被重复回复

### 2. 智能去重
- 基于 MD5 hash 的内容去重
- 支持时间窗口（默认 120 秒）
- 缓存大小可配置（默认 50 条）

### 3. 异常恢复
- OCR 连续失败自动重建
- 网络超时自动重试
- 详细的错误日志记录

### 4. 性能优化
- 局部 OCR（只识别聊天区域）
- 异步飞书同步（不阻塞消息发送）
- 轮转日志（防止日志文件过大）

### 5. 可配置性
- 所有参数都在 `config.py` 中
- 支持 `.env` 环境变量覆盖
- 易于扩展和定制

## 配置说明

### config.py 主要参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `MESSAGE_DEDUP_WINDOW` | 50 | 去重缓存大小 |
| `MESSAGE_DEDUP_TIME_WINDOW` | 120 | 去重时间窗口（秒） |
| `MESSAGE_BBOX_DISTANCE_THRESHOLD` | 50 | 位置变化阈值（像素） |
| `CROP_TOP_RATIO` | 0.15 | 裁剪上边界比例 |
| `CROP_BOTTOM_RATIO` | 0.82 | 裁剪下边界比例 |
| `POLL_INTERVAL_NO_MSG` | 0.5 | 无新消息轮询间隔 |
| `POLL_INTERVAL_MAIN` | 0.5 | 主循环轮询间隔 |

## 故障排查

### 问题 1：识别不到消息

**原因**：OCR 识别失败、消息被过滤、气泡检测失败

**解决**：
- 检查 `logs/lobster.log` 中的 DEBUG 日志
- 调整 `CROP_TOP_RATIO` 和 `CROP_BOTTOM_RATIO`
- 检查气泡颜色范围是否匹配

### 问题 2：重复回复

**原因**：位置追踪阈值太小、去重时间窗口太短

**解决**：
- 增加 `MESSAGE_BBOX_DISTANCE_THRESHOLD`
- 增加 `MESSAGE_DEDUP_TIME_WINDOW`

### 问题 3：飞书同步失败

**原因**：API 密钥错误、表格权限不足、字段名不匹配

**解决**：
- 检查 `.env` 中的密钥
- 确保应用有多维表格权限
- 确认字段名：`用户消息 / User`、`龙虾回复 / Assistant`
