# Project Claw 轻量级联邦学习系统完整指南

## 🎯 系统架构

### 联邦学习流程

```
边缘设备 (Edge Box)                    云服务器 (Cloud Server)
    ↓                                      ↓
每天凌晨 3 点                        联邦中心 Hub
    ↓                                      ↓
1. 提取高质量对话                   1. 接收适配器
   (Satisfaction > 0.8)                  ↓
    ↓                                2. 验证校验和
2. 本地 LoRA 微调                       ↓
   (unsloth/peft)                   3. 存储适配器
    ↓                                      ↓
3. 打包适配器权重                   4. 聚合模型
   (不传输原始文本)                      ↓
    ↓                                5. 生成全局模型
4. 上传到联邦中心                       ↓
   (仅权重 + 元数据)                6. 分发回边缘设备
```

### 第 3 步：创建联邦中心接收端

```python
# cloud_server/federated_hub.py

from fastapi import FastAPI, UploadFile, File, Form
import pickle
import gzip
import logging

app = FastAPI()
logger = logging.getLogger(__name__)

@app.post("/federated_hub/upload")
async def upload_adapter(
    adapter: UploadFile = File(...),
    metadata: UploadFile = File(...),
    device_id: str = Form(...),
    adapter_id: str = Form(...),
    checksum: str = Form(...)
):
    """接收边缘设备上传的适配器"""
    
    try:
        # 读取适配器数据
        adapter_data = await adapter.read()
        metadata_data = await metadata.read()
        
        # 验证校验和
        import hashlib
        calculated_checksum = hashlib.sha256(adapter_data).hexdigest()
        
        if calculated_checksum != checksum:
            return {"success": False, "message": "校验和验证失败"}
        
        # 解压并解析
        decompressed = gzip.decompress(adapter_data)
        package_data = pickle.loads(decompressed)
        
        # 保存适配器
        adapter_path = f"./federated_hub/adapters/{adapter_id}"
        os.makedirs(adapter_path, exist_ok=True)
        
        for file_name, file_data in package_data["adapter_files"].items():
            file_path = os.path.join(adapter_path, file_name)
            os.makedirs(os.path.dirname(file_path), exist_ok=True)
            with open(file_path, "wb") as f:
                f.write(file_data)
        
        # 保存元数据
        metadata_path = f"./federated_hub/metadata/{adapter_id}.json"
        with open(metadata_path, "w") as f:
            f.write(metadata_data.decode())
        
        logger.info(f"✓ 适配器已接收: {adapter_id} from {device_id}")
        
        return {
            "success": True,
            "message": "适配器已接收",
            "adapter_id": adapter_id,
            "checksum": checksum
        }
    
    except Exception as e:
        logger.error(f"接收适配器失败: {e}")
        return {"success": False, "message": str(e)}
```

## 🔧 配置参数

### TrainingConfig

```python
@dataclass
class TrainingConfig:
    model_name: str = "deepseek-ai/deepseek-7b"  # 基础模型
    lora_r: int = 8                              # LoRA 秩
    lora_alpha: int = 16                         # LoRA alpha
    lora_dropout: float = 0.05                   # Dropout
    learning_rate: float = 1e-4                  # 学习率
    num_epochs: int = 3                          # 训练轮数
    batch_size: int = 4                          # 批大小
    max_seq_length: int = 512                    # 最大序列长度
    satisfaction_threshold: float = 0.8          # 满意度阈值
    min_dialogue_pairs: int = 10                 # 最小对话对数
```
