# Project Claw 完整一键启动脚本使用指南

## 🚀 最简单的启动方式

### 双击运行
```
d:\桌面\Project Claw\完整一键启动.bat
```

**自动启动：**
- ✅ Redis 缓存服务（端口 6379）
- ✅ 后端 API 服务（端口 8765）
- ✅ 融资路演大屏（端口 8501）
- ✅ 打开浏览器
- ✅ 显示小程序启动说明

**预计时间：** 2-3 分钟

---

## 📋 启动流程详解

### 第 1 步：环境检查（自动）
```
✓ 虚拟环境检查
✓ requirements.txt 检查
✓ 后端文件检查
✓ 大屏文件检查
✓ 小程序文件检查
```

### 第 2 步：启动 Redis（自动）
```
docker run -d -p 6379:6379 redis:latest

功能：
- 缓存用户画像
- 缓存对话历史
- 缓存订单信息
- 缓存会话数据
```

### 第 3 步：启动后端 API（自动）
```
python -m uvicorn cloud_server.api_server_pro:app --host 0.0.0.0 --port 8765 --reload

功能：
- 处理 C端小程序请求
- 处理 B端商家系统请求
- 管理 Agent 对话
- 管理用户和商家画像
- 管理订单和交易
```

### 第 4 步：启动融资路演大屏（自动）
```
streamlit run cloud_server/god_dashboard.py --server.port 8501

功能：
- B端商家系统
- 显示实时数据
- 显示对话历史
- 显示订单管理
- 显示商家管理
- 显示数据分析
```

### 第 5 步：打开浏览器（自动）
```
自动打开：
- http://localhost:8765/docs（后端 API 文档）
- http://localhost:8501（融资路演大屏）
```

### 第 6 步：启动小程序（手动）
```
1. 打开微信开发者工具
2. 打开项目：d:\桌面\Project Claw\miniprogram
3. 点击"编译"或按 Ctrl+Shift+R
4. 在模拟器中预览
```

### 方式 2：使用 PowerShell 命令

```powershell
Stop-Process -Name python -Force

# 停止所有 streamlit 进程

# 停止 Redis 容器
docker stop $(docker ps -q --filter ancestor=redis:latest)
```

## 💡 常见问题

### Q1：脚本无法执行
```
错误：无法加载文件

解决：
1. 以管理员身份打开 PowerShell
2. 运行：Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
3. 输入 Y 确认
4. 重新运行脚本
```

### Q2：Docker 未安装
```
错误：Docker 未安装或未运行

解决：
1. 下载 Docker Desktop
2. 安装并启动 Docker Desktop
3. 重新运行脚本
```

### Q3：端口被占用
```
错误：Address already in use

解决：
1. 检查是否已有服务在运行
2. 关闭占用端口的程序
3. 或修改端口号
```

### Q4：小程序无法连接
```
错误：服务器连接失败

解决：
1. 检查后端 API 是否运行
2. 检查 API 地址是否正确
3. 检查防火墙设置
4. 重新编译小程序
```
