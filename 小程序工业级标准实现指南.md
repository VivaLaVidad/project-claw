## 🚀 快速启动

### 第 1 步：在微信开发者工具中打开项目

1. 打开微信开发者工具
2. 选择 "打开项目"
3. 选择 `d:\桌面\Project Claw\miniprogram` 文件夹
4. 输入 AppID (如果没有，使用测试 AppID)
5. 点击 "打开"

### 第 2 步：配置服务器地址

编辑 `miniprogram/app.js`，修改服务器地址：

```javascript
globalData: {
  serverBase: 'http://localhost:8765',  // 后端 API 地址
  wsBase: 'ws://localhost:8765'         // WebSocket 地址
}
```

### 第 3 步：启动小程序

在微信开发者工具中：
- 点击 "预览" 或 "真机调试"
- 或直接在模拟器中查看

## 📊 与后端的集成

### API 端点映射

```
小程序请求          →  后端 API
─────────────────────────────────
POST /auth/login    →  /auth/login
GET /auth/user-info →  /auth/user-info
POST /auth/logout   →  /auth/logout
GET /orders/list    →  /orders/list
GET /orders/{id}    →  /orders/{id}
WS /ws/audio/{id}   →  /ws/audio/{id}
```

### 认证流程

```
1. 小程序启动
   ↓
2. 调用 wx.login() 获取 code
   ↓
3. 发送 code 到后端 POST /auth/login
   ↓
4. 后端返回 session_id 和 user_info
   ↓
5. 小程序保存 session_id 到本地存储
   ↓
6. 后续请求自动添加 Authorization header
```

### 错误处理

```
401 Unauthorized
  ↓
清除本地会话
  ↓
重新登录
  ↓
重试请求
```

## 📝 常见问题

### Q: 如何修改后端地址？

A: 编辑 `app.js` 中的 `globalData`:

```javascript
globalData: {
  serverBase: 'http://your-server:8765',
  wsBase: 'ws://your-server:8765'
}
```

### Q: 如何处理网络错误？

A: API 服务层自动处理，会显示错误提示并重试。

### Q: 如何添加新的 API？

A: 在 `api/service.js` 中添加新的 API 方法：

```javascript
const newAPI = {
  getNewData() {
    return apiService.get('/new-endpoint', { showLoading: true })
  }
}

module.exports = {
  // ... 其他 API
  newAPI
}
```

### Q: 如何自定义样式？

A: 编辑对应页面的 `.wxss` 文件，或修改 `app.wxss` 全局样式。
