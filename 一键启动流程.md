# Project Claw 完整一键启动流程

## 🚀 三步启动你的整个项目

### 第 1 步：快速安装依赖（首次运行）
```
双击运行：d:\桌面\Project Claw\快速安装.bat

预期时间：2-5 分钟
自动完成：
  ✓ 创建虚拟环境

```
双击运行：d:\桌面\Project Claw\一键启动.bat

自动完成：
  ✓ 环境检查
  ✓ 启动 Redis（端口 6379）
  ✓ 启动后端 API（端口 8765）

## 🛑 停止所有服务

### 方式 1：关闭命令行窗口
```
关闭所有启动的命令行窗口
```

### 方式 2：使用 PowerShell 命令
```powershell
# 停止所有 Python 进程
Stop-Process -Name python -Force

# 停止所有 streamlit 进程
Stop-Process -Name streamlit -Force

# 停止 Redis 容器
docker stop $(docker ps -q --filter ancestor=redis:latest)
```

## 💡 常见问题

### Q1：虚拟环境不存在
```
解决：双击运行 快速安装.bat
```

### Q2：Docker 未安装
```
解决：
1. 下载 Docker Desktop
2. 安装并启动 Docker Desktop
3. 重新运行一键启动.bat
```

### Q3：端口被占用
```
解决：
1. 检查是否已有服务在运行
2. 关闭占用端口的程序
3. 或修改端口号
```

### Q4：小程序无法连接
```
解决：
1. 检查后端 API 是否运行
2. 检查 API 地址是否正确
3. 重新编译小程序
```
