# Project Claw 一键启动 - 配置清单

## 🚀 一键启动

### 最简单的方式
```
双击运行：d:\桌面\Project Claw\启动.bat
```

**自动启动：**
- ✅ Redis（端口 6379）
- ✅ 后端 API（端口 8765）
- ✅ 融资路演大屏（端口 8501）
- ✅ 小程序启动说明

---

## 📋 启动前需要准备的配置

### 1️⃣ 必需软件（必须安装）
```
✅ Python 3.11+
✅ Docker Desktop
✅ Git
✅ 微信开发者工具
```

### 2️⃣ 虚拟环境（必须创建）
```powershell
cd "d:\桌面\Project Claw"
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

### 3️⃣ 环境变量（可选）
```
创建文件：d:\桌面\Project Claw\.env

内容：
DEEPSEEK_API_KEY=your_api_key
REDIS_URL=redis://localhost:6379
DATABASE_URL=sqlite:///./audit.db
```

### 4️⃣ 端口检查（必须检查）
```powershell
# 检查端口是否被占用
netstat -ano | findstr :8765
netstat -ano | findstr :8501
netstat -ano | findstr :6379

# 如果被占用，修改端口或关闭占用程序
```

## 💡 常见问题

### Q1：虚拟环境不存在
```powershell
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

### Q2：Docker 未安装
```
下载：https://www.docker.com/products/docker-desktop
安装并启动 Docker Desktop
```

### Q3：端口被占用
```powershell
# 查看占用进程
netstat -ano | findstr :8765

# 关闭进程
taskkill /PID <PID> /F
```
