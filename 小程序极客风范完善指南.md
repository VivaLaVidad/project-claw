# Project Claw 小程序极客风范完善指南

## 🎯 架构设计

### Domain-Driven Design (DDD)

```
Bounded Context: MiniProgram
├── Aggregate Root: TradeIntent
│   ├── intent_id (UUID)
│   ├── tenant_id (龙虾盒子 ID)
│   ├── dialogue_type (谈判类型)
│   ├── state (pending → active → completed)
│   └── metadata (扩展数据)
│
├── Entity: DialogueMessage
│   ├── message_id
│   ├── sender (user | agent)
│   ├── content
│   └── confidence
│
└── Value Object: AudioFrame
    ├── frame_id
    ├── data (不可变)
    ├── duration
    └── sample_rate
```

### 六边形架构 (Hexagonal Architecture)

```
┌─────────────────────────────────────────┐
│         业务逻辑核心 (Domain)            │
│  ├─ NegotiationEngine (谈判引擎)        │
│  ├─ StateManager (状态管理)             │
│  └─ EventBus (事件总线)                 │
└─────────────────────────────────────────┘
         ↑                    ↑
    ┌────┴────┐          ┌────┴────┐
    │ 输入端口 │          │ 输出端口 │
    └────┬────┘          └────┬────┘
         ↓                    ↓
    ┌─────────────┐      ┌──────────────┐
    │ WebSocket   │      │ LocalStorage │
    │ HTTP API    │      │ Cache        │
    │ 用户交互    │      │ EventBus     │
    └─────────────┘      └──────────────┘
```

## 📊 数据流向

### 用户交互流程

```
用户操作
  ↓
页面事件处理
  ↓
调用 NegotiationEngine
  ↓
StateManager 更新状态
  ↓
EventBus 发出事件
  ↓
页面监听事件并更新 UI
  ↓
WebSocketAdapter 与后端通信
  ↓
CacheManager 缓存数据
```

### 认证流程

```
应用启动
  ↓
AppBootstrapper.bootstrap()
  ↓
检查本地 sessionId
  ↓
如果存在，验证会话
  ↓
如果无效，执行登录
  ↓
wx.login() 获取 code
  ↓
POST /auth/login
  ↓
保存 sessionId 和 userInfo
  ↓
StateManager 更新状态
  ↓
应用就绪
```

### 对话流程

```
用户点击"语音对讲"
  ↓
NegotiationEngine.createTradeIntent()
  ↓
NegotiationEngine.startDialogue()
  ↓
WebSocketAdapter.connect()
  ↓
开始录音
  ↓
NegotiationEngine.processAudioFrame()
  ↓
WebSocketAdapter.send()
  ↓
后端处理并返回
  ↓
WebSocketAdapter 接收
  ↓
EventBus 发出事件
  ↓
页面更新 UI
  ↓
用户点击"结束对讲"
  ↓
NegotiationEngine.endDialogue()
  ↓
WebSocketAdapter.disconnect()
  ↓
对话完成
```

## 🔧 使用示例

### 在页面中使用

```javascript
// pages/dialogue/dialogue.js

const app = getApp()

Page({
  data: {
    isConnected: false,
    messages: []
  },

  onLoad() {
    // 获取核心模块
    this.stateManager = app.getStateManager()
    this.eventBus = app.getEventBus()
    this.negotiationEngine = app.getNegotiationEngine()

    // 订阅状态变化
    this.unsubscribeDialogueState = this.stateManager.subscribe(
      'dialogueState',
      (state) => {
        this.setData({ isConnected: state === 'active' })
      }
    )

    // 监听事件
    this.eventBus.on('ws:message', (message) => {
      this.handleMessage(message)
    })
  },

  async startDialogue() {
    try {
      // 创建谈判意图
      const intent = await this.negotiationEngine.createTradeIntent(
        'tenant_001',
        'negotiation'
      )

      // 启动对话
      await this.negotiationEngine.startDialogue(intent)

      this.setData({ isConnected: true })
    } catch (error) {
      console.error('启动对话失败:', error)
      app.showToast('启动失败', 'error')
    }
  },

  async stopDialogue() {
    try {
      await this.negotiationEngine.endDialogue()
      this.setData({ isConnected: false })
    } catch (error) {
      console.error('停止对话失败:', error)
    }
  },

  handleMessage(message) {
    const messages = this.data.messages
    messages.push(message)
    this.setData({ messages })
  },

  onUnload() {
    // 取消订阅
    this.unsubscribeDialogueState?.()
    
    // 移除事件监听
    this.eventBus.clear('ws:message')
  }
})
```
