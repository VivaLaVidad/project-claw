## 🔍 代码逻辑走通验证

### 第 1 步：谈判启动流程

```
1. 用户在小程序发起询价
   ↓
2. 后端创建 NegotiationOrchestrator
   ↓
3. 创建 ClientAgent 和 MerchantAgent
   ↓
4. 启动谈判会话（session_id）
   ↓
5. C端Agent发送初始INQUIRY消息
```

**验证：✅ 逻辑完整，无缺失**

### 第 2 步：B端Agent处理流程

```
1. B端Agent收到INQUIRY消息
   ↓
2. 根据策略计算报价
   - aggressive: base_price * 0.98
   - conservative: base_price * 1.05
   - normal: base_price
   ↓
3. 检查最低价格底线
   - 如果expected_price < min_price，调整报价
   ↓
4. 生成OFFER消息
   ↓
5. 返回给C端Agent
```

**验证：✅ 逻辑完整，价格保护有效**

### 第 3 步：C端Agent处理流程

```
1. C端Agent收到OFFER消息
   ↓
2. 计算价格差异
   - diff = (offered - expected) / expected
   ↓
3. 根据差异决策
   - diff <= 5%：接受（ACCEPTANCE）
   - 5% < diff <= 15%：反价（COUNTER_OFFER）
   - diff > 15%：拒绝（REJECTION）
   ↓
4. 生成相应消息
   ↓
5. 返回给B端Agent
```

**验证：✅ 逻辑完整，决策清晰**

### 第 4 步：反价协商流程

```
1. B端收到COUNTER_OFFER
   ↓
2. 检查反价是否可接受
   - counter_price >= min_price：接受
   - counter_price < min_price：继续反价
   ↓
3. 如果继续反价，计算新价格
   - new_price = (counter + min_price) / 2
   ↓
4. 生成COUNTER_OFFER消息
   ↓
5. 返回给C端Agent
```

**验证：✅ 逻辑完整，价格收敛**

### 第 5 步：协议达成流程

```
1. 任一方发送ACCEPTANCE消息
   ↓
2. 对方收到并确认
   ↓
3. 生成FINAL_DECISION消息
   ↓
4. 谈判状态变更为AGREEMENT_REACHED
   ↓
5. 返回最终价格和订单ID
```

**验证：✅ 逻辑完整，协议确认有效**

## 🔧 集成步骤

### 第 1 步：在 api_server_pro.py 中导入

```python
from cloud_server.agent_system import NegotiationOrchestrator

# 创建全局协调器
orchestrator = NegotiationOrchestrator()
```

### 第 2 步：创建API路由

```python
@app.post("/api/v1/negotiation/start")
async def start_negotiation(request: Dict):
    """启动谈判"""
    session_id = await orchestrator.start_negotiation(
        client_id=request["client_id"],
        merchant_id=request["merchant_id"],
        item=request["item"],
        price=request["price"],
        client_profile=request.get("client_profile"),
        merchant_profile=request.get("merchant_profile")
    )
    return {"session_id": session_id}

@app.post("/api/v1/negotiation/run")
async def run_negotiation(request: Dict):
    """运行谈判"""
    session_id = request["session_id"]
    success, final_price = await orchestrator.run_negotiation(session_id)
    return {
        "success": success,
        "final_price": final_price,
        "history": orchestrator.get_history(session_id),
        "status": orchestrator.get_status(session_id)
    }

@app.get("/api/v1/negotiation/{session_id}/history")
async def get_history(session_id: str):
    """获取谈判历史"""
    return {"messages": orchestrator.get_history(session_id)}

@app.get("/api/v1/negotiation/{session_id}/status")
async def get_status(session_id: str):
    """获取谈判状态"""
    return orchestrator.get_status(session_id)
```

### 第 3 步：小程序调用

```javascript
// 启动谈判
async function startNegotiation() {
  const res = await wx.request({
    url: 'http://localhost:8765/api/v1/negotiation/start',
    method: 'POST',
    data: {
      client_id: 'client_001',
      merchant_id: 'merchant_001',
      item: 'iPhone 15',
      price: 5000,
      client_profile: {
        price_sensitivity: 0.8
      },
      merchant_profile: {
        strategy: 'normal'
      }
    }
  })
  
  const sessionId = res.data.session_id
  
  // 运行谈判
  const negotiationRes = await wx.request({
    url: 'http://localhost:8765/api/v1/negotiation/run',
    method: 'POST',
    data: { session_id: sessionId }
  })
  
  console.log('谈判结果:', negotiationRes.data)
}
```
